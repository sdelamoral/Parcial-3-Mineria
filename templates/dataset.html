Dataset

El dataset proviene de Amazon Kindle Reviews (Kaggle) y contiene 982,619 reseñas originales. Para este proyecto se trabajó con una muestra de 50,000 reseñas usando solo las columnas reviewText y overall.

Las calificaciones se convirtieron a sentimiento binario: ratings 1-2 → Negativo (0), ratings 4-5 → Positivo (1), ratings 3 descartados por ambiguos.

Desbalance de clases
El problema más importante del dataset es el desbalance severo: el 93.4% de las reseñas son positivas y solo el 6.6% son negativas (ratio 14:1). Un modelo que siempre prediga "Positivo" alcanzaría 93.4% de accuracy sin aprender nada útil. Por eso se usó F1-score como métrica principal.
